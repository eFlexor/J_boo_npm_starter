# SalesChatGPT - Inventory Manager

Offline-first inventory management app for resellers. No backend required.

[![CI](https://github.com/eFlexor/J_boo_npm_starter/workflows/CI/badge.svg)](https://github.com/eFlexor/J_boo_npm_starter/actions)

## Features

- Dashboard KPIs: inventory value, cost basis, profit, revenue
- Full inventory CRUD with 15 fields and auto-generated SKUs
- Image gallery per item (resized, stored in IndexedDB)
- Sales recorder with automatic profit calculation
- CSV import/export and JSON backup/restore
- Search, filter, sort across all fields
- Dark mode, offline-first

## Getting started

    git clone https://github.com/eFlexor/J_boo_npm_starter
    cd J_boo_npm_starter
    npm install
    npm start

Open http://localhost:3000
